Come ottenere le credenziali:
1. Vai su supabase.com → crea un progetto
2. Settings → API → copia Project URL e anon key
3. Esegui l'SQL nel tab qui sotto per creare le tabelle
📋 Mostra SQL da eseguire in Supabase
-- 1. Tabella clienti Sisal create table if not exists sisal ( id uuid default gen_random_uuid() primary key, nome text, tel text, email text, cf text, piattaforma text, prodotto text, data_attivazione date, commissione numeric, commissione_ricorrente numeric, stato text, tipo_conto text, priorita text, note text, created_at timestamptz default now() ); -- 2. Tabella allegati (per tutte le sezioni) create table if not exists allegati ( id uuid default gen_random_uuid() primary key, sezione text, record_id text, nome_file text, url text, tipo text, created_at timestamptz default now() ); -- 3. Storage bucket per file insert into storage.buckets (id, name, public) values ('allegati', 'allegati', true) on conflict do nothing; -- 4. Policy accesso pubblico create policy "Public Access" on storage.objects for all using (bucket_id = 'allegati');
